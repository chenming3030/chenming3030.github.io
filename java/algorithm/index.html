# 引言

数据结构和算法 是计算机专业必修的一门课程，它非常的重要，是编程必备的基础，但是这门课程不太好学习，因为学习它会感到非常的枯燥乏味。


但是，如果你想让自己的编程能力有质的飞跃，不再停留于调用别人封装好的API，想要追求更完美的实现，那么就必须学习数据结构和算法，因为 程序设计=数据结构+算法。


通过对数据结构和算法的学习，能让你更深层次的理解程序，提升你的编码能力，让你编写的代码更优雅，性能更高。
