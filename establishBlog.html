# 本文档搭建指南

# 准备工作

  1. nodejs v14.0.0

官网：http://nodejs.cn/

  1. git 2.36.1

官网：https://git-scm.com/

注意：git版本一定要新，不然将文件push到github上时，使用密码认证总是报错失败。因为github自2021年8月13日移除了密码认证的支持，建议使用 personal access token 代替密码认证。

  1. VuePress ^1.9.7

官网：https://www.vuepress.cn

  1. GitHub Pages

官网：https://pages.github.com

# 搭建过程

具体搭建过程参考 VuePress 官网即可

  1. 安装nodejs
  2. 安装git
  3. 创建 GitHub Pages
  4. 初始化 VuePress 脚手架，并根据自身需求修改配置
  5. 编译文档，将生成的 docs\.vuepress\dist 下的所有文件push到github上
